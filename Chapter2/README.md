作业第一个层次，主要是要求在考虑风阻以及空气密度随海拔的变化对弹道区县的影响，并且求出在哪个角度下发射能使炮弹飞的最远。
作业的第二个层次是要求开发一套精确打击目标的程序，所给的目标是与炮弹有高度差，要求目标处于炮弹的轨迹上，并求出能够打击目标的最小发射角。
