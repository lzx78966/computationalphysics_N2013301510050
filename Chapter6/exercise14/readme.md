>##摘要
一维波动方程利用差分法来求解，验证在一根两端边界固定的弦在初始时刻给一个扰动，之后的过程中，产生的两个波包，在碰撞后，速度以及振幅不变的事实。

>##结论
从上图中可以看出两个波包在碰撞到边界后，发生半波损失，振幅反向，且可以看出，两个波包在碰撞前后，相速以及振幅都不变，各自沿着原来的方向运动。
