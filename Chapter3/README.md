>##结论
经过观察，可以发现，在简谐振动中，运动周期与振幅大小无关(此处振幅的大小取决于初速度)，而非简谐振动的情况下，振幅越小，振动周期越大，即两者是存在某种联系的。 对于简谐振动$\omega$=$\sqrt{k/m}$.而对于这里的非简谐振动，通过maple解得解析解为Jacobi正弦函数:
